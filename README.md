# raycast

My personal collection of custom [Raycast](https://raycast.com) scripts and extensions.

## Scripts

Point Raycast to the `scripts/` folder via **Settings → Extensions → Script Commands → Add Folders**.

| Script | Description |
|--------|-------------|
| `count-weekdays.py` | Count how many times a weekday appears in a month. E.g. _"saturdays this month"_, _"fridays in august 2027"_ |
